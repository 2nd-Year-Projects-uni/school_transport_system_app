# Environment Variables

All sensitive API keys and tokens are now stored in a `.env` file at the project root. This file is listed in `.gitignore` and should never be committed to the repository.

**How to use:**
- Copy `.env.example` to `.env` and fill in your own secret values.
- Never share your `.env` file publicly or commit it to git.

**Note:**
If you are a new collaborator, ask the project owner for the required environment variable values.
